Installation:

1. Preparing the environment

ToeOS can be installed from a virtually any existing Linux installation (including the live environments) as long as it contains the following dependencies:

`wget tar make gcc...`

We recommend distributions that are already using musl libc, as compiling the packages that way is going to make them smaller.

2. Partitioning the disks

Pick your favourite disk-partitioning tool, create a root partition for your fresh ToeOS install and create any other partitions you might need.

It is possible to install ToeOS to a directory on your system, if you that is what you want then the second step can be skipped.

3. Running the installation script

Run the following command in a directory on your host system:

`git clone ...`

Then move to the created directory and edit the installation script to your liking. The main things you might want to change are the keyboard layout, timezone and most importantly, the fstab file. After that you can begin the installation by running the script.

4. Finishing up

The amount of time it takes to build ToeOS depends on your hardware, when it is finished you can boot into the newly installed system. As long as you are on a wired connection you should be able to connect to the internet and manually build anything you want.
